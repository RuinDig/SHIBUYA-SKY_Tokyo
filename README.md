# SHIBUYA-SKY_Tokyo
渋谷スクランブルスクエアの屋上展望施設SHIBUYA SKY(渋谷スカイ)。14階～45階の移行空間「SKY GATE」、屋外展望空間「SKY STAGE」、46階の屋内展望回廊「SKY GALLERY」の3つのエリアで構成されている。【場所：東京都渋谷区】
